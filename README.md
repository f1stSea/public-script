# public-script

Public deployment scripts, auto-synced from [evergreen](https://github.com/f1stSea/evergreen) via GitHub Actions.

## Files

| File | Description |
|------|-------------|
| `deploy.sh` | Evergreen Dashboard one-click deploy/update |
| `docker-compose.yml` | Evergreen Dashboard compose configuration |

## Usage

```bash
curl -fsSL https://raw.githubusercontent.com/f1stSea/public-script/main/deploy.sh | bash
```
